This part is without user interface, to run :
- enter in your terminal : python 1 Register your face.py

Webcam window open and take 30 picture of you every 100 millisecond and save in a file.

- enter in your terminal : python 2 Learning your face.py
Create an average of your face.

- enter in your terminal : python 3 Detect.py
Webcam window open recognize your face and print your id.

You need to have  OpenCV, Pillow, install
All data base is located in « trainner » file in trainner.yml
